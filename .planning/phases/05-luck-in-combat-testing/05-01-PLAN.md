From js/mechanics.js (internal helper — postAction):
```javascript
async function postAction(bookNumber, actionType, details) {
    try {
        const res = await fetch(`/api/sessions/${bookNumber}/actions`, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ action_type: actionType, details }),
        });
        if (!res.ok) return null;
        return await res.json();
    } catch {
        return null;
    }
}
```

4. Uses `damage_before - damage_after` formula: positive for lucky (restores stamina), negative for unlucky (deducts more)
